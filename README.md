# One Screen Portfolio

A modular React portfolio screen built with Vite.

## Structure

- `src/features/home-screen` contains the portfolio screen and its smaller components.
- `src/assets/icons` stores SVG icon assets.
- `src/assets/images` stores visual image assets.
- `src/assets/fonts` stores font styling entry points.

## Scripts

```bash
npm install
npm run dev
npm run build
```
